# Dream Board Mobile — Official Terminology

This document defines the official language of Dream Board Mobile. All contributors, collaborators and builders must use this terminology consistently across code, UI, ARIA labels, documentation and communications. No substitutions.

**The official terms:**

**Dream** — not Post

**ReDream** — not Repost

**Circle** — not Group or Community

**Dreamor** — not User or Member

**Why this matters:** Dream Board Mobile is built on the belief that language shapes experience. Every word in this app was chosen deliberately.

---

## About this project

Dream Board Mobile was built by Alhasan, a blind Dreamor, with Claude (Anthropic's AI) as his development partner. Accessibility is not a feature here — it is the foundation. This project exists to prove that the right tools and the right collaboration can build something meaningful regardless of what others assume about what's possible.
